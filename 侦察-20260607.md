- **我们的判断**：亿航首次季度盈利是里程碑事件——证明 eVTOL 不仅是"概念"，而是可以赚钱的生意。空域管理改革正在加速（十部门标准体系 + 五部门低空基础设施意见），瓶颈在解决而非固化。低空经济是少有的"政策+产业+业绩"三重验证的方向
- **潜在空间**：2026-2030 年复合增速 52.9%，万亿→3.5 万亿，产业链标的处于"从 0 到 1"阶段
- **证伪条件**：空域管理改革停滞、eVTOL 安全事故、亿航后续季度重回亏损

### 🟡 A 级：AI Agent/智能体

**一句话逻辑**：2026 是 Agent 商业化元年，Salesforce Agentforce ARR 8 亿（+169%）、Microsoft 365 Copilot 2000 万付费席位，但纯 A 股映射有限。

**三大映射信号**：
- **政策**：网信办《智能体规范应用与创新发展实施意见》（2026-05）
- **海外**：Microsoft Build 7 款自研模型 + NVIDIA Agent Toolkit + CB Insights 报告（50% 融资热点来自 Agent）
- **一级**：月之暗面 20 亿美元（估值 200 亿）+ DeepSeek 500 亿人民币（估值 600 亿美元）+ 每 5 家新独角兽 1 家做 Agent

**五维验证**：

| 维度 | 评分 | 关键数据 |
|------|:----:|------|
| 渗透率拐点 | **3** | AI Agent 企业采用率快速提升（2026 多数企业 ROI 可衡量），但仍处早期，从"试点"到"规模化"跨越中 |
| 产能周期 | **3** | 软件属性，不适用传统产能框架，取中性 |
| 量价热度 | **3** | AI 概念整体中高热度，A 股缺乏纯 Agent 标的，热度分散在 AI 应用/软件板块 |
| 资金流向 | **3** | 一级市场极度活跃，二级市场 AI 应用 ETF 有资金流入但不够聚焦 |
| 催化事件 | **3** | Microsoft Build 刚过、CES Asia AI 专场、阿里腾讯 AI 战略发布，催化密集但偏海外 |
| **验证总分** | **15/25** | — |

**综合评分**：

```
映射总分：13/15（权重 40%）
验证总分：15/25（权重 60%）
综合得分 = 13/15 × 0.4 + 15/25 × 0.6 = 0.347 + 0.360 = 0.707
评级：🟡 A 级
```

**核心问题**：Agent 无疑是 2026 年最大的科技主题之一，但 **A 股缺乏纯正 Agent 标的**。Agent 的投资机会主要在美股（Salesforce、Microsoft、ServiceNow）和港股（阿里巴巴、腾讯）。A 股映射更多是通过 AI 算力（光模块/服务器）、AI 应用（办公/客服软件）间接参与，确定性不如人形机器人和低空经济。

**结论**：Agent 方向正确，但 **A 股配置优先级低于人形机器人和低空经济**。建议通过人形机器人（具身智能是 Agent 的物理化）+ AI 半导体设备（Agent 算力底座）间接参与。
